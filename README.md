## Lancement
Pour pouvoir jouer, il est nécessaire d'avoir un serveur KOT ouvert sur un réseau connu.

Soit avec un serveur distant, soit un serveur local (localhost)
Il suffit juste de lancer un ServeurKoT.exe sur la machine hôte.

Chaque Joueur doit lancer une instance de ClientKoT.exe.

Il se connecte le serveur ou il peut ajouter un serveur.
Il peut après lancer une partie.
